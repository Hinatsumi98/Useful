### Windows Installs.

 1. Install [Visual C++ x64](https://github.com/abbodi1406/vcredist/releases/).
 2. Install [.NET Runtime 9.X.X Desktop x64](https://dotnet.microsoft.com/en-us/download/dotnet/9.0).
 3. Install [.NET 6.0 Runtime Desktop x64](https://dotnet.microsoft.com/en-us/download/dotnet/thank-you/runtime-desktop-6.0.30-windows-x64-installer).
 4. Install [Discord](https://discord.com/api/downloads/distributions/app/installers/latest?channel=stable&platform=win&arch=x64)
 5. Install [Steam](https://cdn.akamai.steamstatic.com/client/installer/SteamSetup.exe)
 6. Install [Firefox](https://download.mozilla.org/?product=firefox-msi-latest-ssl&os=win64&lang=en-US)
 7. Install [Spotify](https://download.scdn.co/SpotifySetup.exe)
 8. Install [AMD Drivers](https://www.amd.com/en/support/download/drivers.html)
 9. To be added.
